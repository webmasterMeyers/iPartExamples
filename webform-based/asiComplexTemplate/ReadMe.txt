To use the Visual Studio Project Template for iParts:

- Place the iPartVSTemplate.zip template file in the Visual Studio project template directory. By default, this directory is "My Documents\Visual Studio 2008\Templates\ProjectTemplates\Visual C#".
- Now, when you create a project in Visual Studio, "iMIS iPart" will be an option under Visual C# templates.
- You Must start your project with a capitol letter so that the class name will be valid

The zip file can be downloaded from here http://www.advsol.com/ASI/DeveloperResources/Best_practices/iParts/Visual_Studio_iPart_template/Developer_Resources/iParts_Development/Visual_Studio_iPart_Template.aspx
